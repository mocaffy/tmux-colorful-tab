## これはなに
Tmux のタブ (window-status) の高度な装飾を可能にするプラグインです。  

## インストール
tmux.conf
```
set -g @plugin 'mocaffy/tmux-tabicon'
run '~/.config/tmux/plugins/tpm/tpm'
```
